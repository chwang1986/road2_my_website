# road2_my_website

学习建立个人网站的过程记录。

## 项目概述

- 项目名称: road2_my_website
- 技术栈: Django (后端) + Vue (前端)
- 目标: 在本地电脑上部署一个个人独立网站，并将项目发布到 GitHub
- 记录目的: 记录开发过程中的步骤、问题和解决方案.

开发环境

操作系统: [Windows]

工具版本:
- Python: [3.13.3]
- Node.js: [22.16.0]
- npm: [10.9.2]
- Git: [2.45.2.windows.1]


项目结构
```
road2_my_website/
├── frontend/ # Vue 前端项目
├── backend/ # Django 后端项目
├── docs/ # 开发文档，从项目初始化开始
└── README.md # 项目说明
```
